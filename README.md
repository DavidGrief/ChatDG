# ChatDG

**ChatDG** — плагин для управления игровым чатом на Bukkit / Spigot / Paper / Purpur.

**Автор:** Давид Grief  
**Версия:** 1.0.3

## Возможности

- локальный и глобальный чат;
- личные сообщения и `/r`;
- муты;
- игнорирование игроков;
- очистка чата;
- антиспам и антикапс;
- фильтр запрещённых слов;
- упоминания игроков;
- HEX / RGB и стандартные `&` цвета;
- поддержка LuckPerms и PlaceholderAPI;
- Hover / Click для сообщений;
- поддержка старых и новых версий Bukkit / Spigot / Paper.

## Установка

1. Поместите `ChatDG.jar` в папку `plugins`.
2. Запустите сервер.
3. Настройте файлы в `plugins/ChatDG/`.
4. После изменения настроек используйте `/chatdg reload`.
